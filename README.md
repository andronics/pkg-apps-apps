# Application Launcher - Quick application access

**Package**: `@apps/apps`
**Version**: 0.1.0
**Repository**: `pkg-apps-apps`

## Overview

Application Launcher - Quick application access

## Installation

### Prerequisites

This package requires the following external commands:

- `dmenu`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @apps/apps

# Or install from local source
cd ~/.pkgs
stow apps
```

## Usage

### Application

Application Launcher - Quick application access

Run the application to see usage instructions.

## Configuration

Configuration files are typically stored in:
- `~/.config/apps/` - User configuration
- `~/.local/share/apps/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @apps/apps

# Or manual unstow
cd ~/.pkgs
stow -D apps
```
